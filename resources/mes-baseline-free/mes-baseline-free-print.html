// WHAT YOU NEED

  • a TradingView account (free or paid)
  • the ORB v2.2 [ORB Alpha] - Free script added to your chart
  • approximately 5 minutes of setup time

// STEP 1 — SET THE SYMBOL

at the top-left of your TradingView chart, change the symbol to: MES1!

this is the continuous front-month contract for Micro E-mini S&P 500 futures. make sure you're on the futures symbol, not the index.

// STEP 2 — SET THE CHART TIMEFRAME

at the top of the chart, set the timeframe to: 5m

(5-minute bars). all baseline numbers are computed on this timeframe.

// STEP 3 — ADD THE INDICATOR

if you haven't already, click the Indicators button at the top of the chart, search for "ORB v2.2 [ORB Alpha] - Free" by ORBAlpha, and add it to your chart.

// STEP 4 — OPEN INDICATOR SETTINGS

click the gear icon next to the indicator name in the chart legend to open the settings dialog.

// STEP 5 — SET SESSION TIMES

  • session start: 09:00
  • session end: 10:00
  • ORB cutoff: 09:35
  • exit time: 16:00
  • timezone: America/New_York (NY session)

these are the indicator's default settings.

// STEP 6 — SET RISK PARAMETERS

  • target risk: $250
  • max risk per trade: $350
  • max contracts: 40

these are the default risk values for micro futures.

// STEP 7 — VERIFY ENTRY MODE

confirm the entry mode is set to: ORB

(no Stretch — Stretch is a Pro-only feature.)

// STEP 8 — SET THE DATE RANGE

set the range to: January 1, 2019 – December 31, 2025

// STEP 9 — OPEN THE STRATEGY TESTER

at the bottom of the chart, click the Strategy Tester tab.

// TROUBLESHOOTING

  1. SYMBOL — confirm MES1!, not ES1! or MNQ1!
  2. TIMEFRAME — confirm 5m
  3. DATE RANGE — confirm January 1, 2019 through December 31, 2025
  4. SESSION TIMES — confirm 09:00 / 10:00 / 09:35 / 16:00 / America/New_York
  5. RISK PARAMETERS — confirm $250 / $350 / 40
  6. ENTRY MODE — confirm "ORB" only
  7. INDICATOR VERSION — confirm "ORB v2.2 [ORB Alpha] - Free"
  8. DEFAULT SETTINGS — try clicking "Defaults" and re-applying

if all checks pass and output still doesn't match, try removing and re-adding the indicator. if it still doesn't match, contact ORBAlpha.
